# ✋ Virtual Air Whiteboard | OpenCV + MediaPipe

## 📌 Project Overview
The **Virtual Air Whiteboard** is a real-time computer vision project that allows users to draw in the air using hand gestures.  
It uses **MediaPipe hand tracking** and **OpenCV** to detect finger positions and convert them into smooth drawing strokes on a digital canvas.

This project demonstrates:
- hand gesture recognition
- real-time drawing
- computer vision
- UI overlay design
- gesture-based erase and undo

---

## 🚀 Features
- ☝️ Draw using index finger
- ✌️ Pause drawing with two fingers
- ✊ Stop drawing with fist gesture
- 👋 Wave hand to erase all drawings
- 🎨 Multiple brush colors
- 📏 Adjustable brush size
- ↩️ Undo functionality
- 🖥️ Real-time webcam drawing board

---

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- Computer Vision
- Hand Landmark Detection
